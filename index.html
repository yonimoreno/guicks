🔒 Privacy Policy

Your data stays on your device

Guicks is designed with privacy as a core principle. We do not store any of your personal data on any server or cloud. Everything you save — bookmarks, favorites, preferences, calculator history, watchlists, saved locations, and translation history — lives exclusively in your browser's local storage on your own device.

What we store locally

The following data is stored only on your device using your browser's localStorage API and is never transmitted to us:

• Saved bookmarks and custom categories
• Dock layout preferences
• Calculator history and favorites
• Currency & unit converter preferences
• Weather saved locations
• Translation history and favorites
• Stock market watchlist
• Lucky app entries and history
• Theme and interface preferences

Analytics

We use Google Analytics to collect anonymized, aggregated usage data (page views, session duration, general location by country). This data contains no personally identifiable information and is used solely to understand how tools are being used so we can improve them. You can opt out via your browser's Do Not Track setting.

No accounts required

Guicks requires no registration, login, or account of any kind. There is no user profile, no email address collected, and no way to identify you as an individual.

Third-party services

Some tools fetch data from external APIs (Open-Meteo for weather, MyMemory for translation, Finnhub for stocks, ExchangeRate-API for currencies). These requests may include your IP address as part of the standard HTTP request, but Guicks does not pass any personal data to these services beyond what is inherent in a browser network request.

Data export & deletion

You can export all your local data at any time using Settings → Save Data to File. You can permanently delete all stored data using Settings → Clear All App Data. Clearing your browser's site data will also remove everything.
