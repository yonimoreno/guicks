Free World Clock — Time Zones, Stopwatch, Countdown & Alarms

Check the current time anywhere in the world, run a stopwatch, set a countdown timer, or create an alarm — all in one free browser app. No download, no sign-up, works on any device.

World time zones

See the current local time in 38 major cities across every time zone — including New York, London, Paris, Dubai, Mumbai, Tokyo, Sydney and more. Essential for remote teams, international calls and travel planning. Each city shows the local time and UTC offset.

Stopwatch and lap timer

A precise stopwatch with lap recording — tap lap to record split times while the clock keeps running. Useful for workouts, cooking, presentations or any timed activity where you need to track intervals.

Countdown timer

Set a countdown for any duration and get a sound alert when time is up. Great for study sessions (Pomodoro technique), cooking, meetings, or any time-boxed activity.

Alarms

Set multiple alarms with custom labels. Alarms use the Web Audio API to play a sound directly in the browser — no notification permissions needed. Keep the tab open for alarms to fire.
